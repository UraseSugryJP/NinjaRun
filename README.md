### Windows
> **Unityプロジェクト参加手順（Win）**
> 1. Git LFSをインストールする（`git lfs install`）
> 2. リポジトリを `git clone` する
> 3. Unity Hubで開く（バージョンは 6.3.x を使用）
> 4. **重要：** もし画像がピンクだったりモデルが消えていたら、ターミナルで `git lfs pull` を実行
> 5. **重要：** Project Settingsの `Asset Serialization` が `Force Text` になっているか
